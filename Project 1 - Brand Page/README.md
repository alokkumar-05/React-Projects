# React Brand Page

This project is a simple **brand landing page** built using **React + Vite**.
The goal of this project was to create the page layout using **React components only**.

## Features
- Responsive Navbar
- Hero Section
- Component-based structure
- Clean UI layout
- Built using Vite for fast development

## Tech Stack
- React
- Vite
- CSS

## Project Structure
- `components/Navigation.jsx`
- `components/HeroSection.jsx`
- `App.jsx`
- `App.css`

## How to Run Locally
```bash
npm install
npm run dev
